## プルリクエスト

略語で PR
またはマージリクエストという（MR）

sub2 で作成、編集 → コミットをした後 sub2 だけプッシュ
Git に移動すると pull request の欄が出ている

1. Reviewers
   レビュワーを選択する(ユーザー名)

2. Assignees
   担当者

3. labels
4. Projects

プルリクの画面の Assignee で自分を選択すると自分のしか出なくなる
プルリクの部分を選択して closed を押すと過去のプルリクエストを閲覧できる
